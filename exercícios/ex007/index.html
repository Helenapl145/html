Formatação de texto

Em um texto pode colocar uma palavra em negrito com a tag b dessa forma
ou com a tag strong dessa forma

Também pode colocar o texto negrito com a tag i assim
ou a tag em dessa forma que é a correta

Tem outras tags para formatação como mark para marcar o texto
del para deixar o texto deletado
small para deixa o texto pequeno
ja para deixar o texto sumbliado pode usar a tag u ou a tag ins que é a correta
por último para formatar textos com exponisação usa a tag sup como: x2 = 25
e para texto como formulas usa a tag sub por exemplo: H2O
